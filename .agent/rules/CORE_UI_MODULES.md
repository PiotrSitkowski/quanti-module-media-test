## modal-message — Built-in System Modal Module

The Quanti Kernel ships a `modal-message` system module registered in the module registry.
Use it for ALL user notifications, confirmations, and error messages.

### Usage via dispatchQuantiEvent (preferred for components)

```tsx
import { dispatchQuantiEvent } from '@quanti/kernel/events.js';

// Show an error modal — NEVER use alert()
dispatchQuantiEvent('quanti:modal:show', {
    type:    'error',
    title:   t.errorTitle,
    message: t.errorText,
});

// Show a confirmation dialog — NEVER use confirm()
dispatchQuantiEvent('quanti:modal:confirm', {
    title:    t.confirmTitle,
    message:  t.confirmMessage,
    onAccept: () => handleDelete(id),
});
```

### Usage via context.api (when dispatching from deep inside a callback)

```tsx
props.context.api?.modal?.show({
    type:    'error',
    title:   t.errorTitle,
    message: errorMessage,
});
```
